BJJFighterBot

I am blue belt Brazilian Jiu-Jitsu practitioner and wanted to use something I'm passionate about for my first real project during my software development journey so I created BJJFighterBot. BJJFighterBot is a reddit bot for the r/BJJ subreddit that responds with a link and description of the fighter called using the keyphrase '!bjjfighterbot'. I had an MVP working in a few hours using Python and deployed it to Heroku.
